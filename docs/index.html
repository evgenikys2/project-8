WHOOP OAuth helper pages

This tiny public site exists only to support personal testing of a WHOOP integration. It provides a privacy policy page and a secure OAuth callback bridge that forwards the WHOOP authorization result to a local development server.

  • Privacy policy URL for WHOOP dashboard setup
  • HTTPS callback page that forwards to http://localhost:8000/auth/callback
  • No data is stored on this static site
Privacy Policy Use this URL in the WHOOP app settings. OAuth Callback Bridge Use this URL as the WHOOP redirect URI.
