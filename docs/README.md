# Documentation Hub

This repository is organized as a personal-first, public-safe project that is preparing for open-source visibility.

## Start Here

If you only read three documents, read these first:

1. [`PROJECT_STRATEGY.md`](./PROJECT_STRATEGY.md)
2. [`ROADMAP.md`](./ROADMAP.md)
3. [`PRIVACY_AND_SECURITY_MODEL.md`](./PRIVACY_AND_SECURITY_MODEL.md)

## Strategy Documents

- [`PROJECT_STRATEGY.md`](./PROJECT_STRATEGY.md): vision, product direction, principles, and success criteria
- [`TECHNICAL_STRATEGY.md`](./TECHNICAL_STRATEGY.md): API, architecture, data, and platform direction
- [`DEPLOYMENT_STRATEGY.md`](./DEPLOYMENT_STRATEGY.md): how the project moves from local use to secure public deployment
- [`AI_INTEGRATION_STRATEGY.md`](./AI_INTEGRATION_STRATEGY.md): how WHOOP context should reach the assistant layer, including mobile-first usage

- [`ROADMAP.md`](./ROADMAP.md): phased execution plan from today's baseline to friend-ready reuse
