Privacy Policy for Project 8

Project 8 is a personal-use application created to test integration with the WHOOP API. It is operated by Evgeny Skoropisov for individual development and testing.

Data Access

After you authorize the application through WHOOP OAuth, the app may access selected WHOOP account data such as profile details, recovery metrics, sleep records, and workout records, depending on the scopes you approve.

How Data Is Used

The data is used only to verify API connectivity, inspect responses, and test local software behavior. The goal is personal development and integration testing, not resale, advertising, or broad analytics.

Storage

OAuth tokens and fetched responses may be stored locally on the operator's development machine for testing purposes. The static pages published here do not store WHOOP account data.

Sharing

WHOOP data accessed by the application is not intentionally sold or shared with third parties except when required for the operation of infrastructure used directly by the operator, such as GitHub Pages for this static privacy page.

Revoking Access

You can revoke the application's access at any time through your WHOOP account settings or by stopping use of the application and deleting local token files.

Contact

For questions about this test application or its handling of data, contact the app operator using the support email configured for the deployment.
